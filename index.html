l 지원 분야와 관련하여 전문성을 키우기 위해 노력한 경험

제가 초반에 한 프로젝트들과 부트캠프에서한 프로젝트 그리고 지금 현재 진행중인 프로젝트에 관해 말씀드리고자 합니다.

일단 초반 프로젝트로 작성한것은 프론트엔드를 주로 다룬 웹사이트들입니다. 노마드코더를 통해 여러 클론코딩을 하였으며 모두다 현재 깃허브에 올린 상태입니다. 리액트에 대한 지식이 많이 부족하여, 한가지가 아닌 여러 리액트를 활용한 클론코딩을 계속해왔습니다. 그 예시로는 트위터 클론코딩, 얼굴 인식 동물상 그리고 NFT 웹사이트등 이 있습니다.

부트캠프에서 web application 을 중심으로 Spring Framework 을 통하여 최종 프로젝트인 병원 게시판을 만들었습니다. 허나 자바의 쓰임새는 여러 종류이며, 자바에 대한 이해를 높이기위해 android studio을 통한 날씨 앱도 만들어보았습니다.

그리고 지금 현재, 요즘 트렌드인 Web 3.0에 대한것과 AWS 자격증과 관련 된 강의를 듣고 클론코딩 하고 있습니다. 저는영어가 편해서 주로 AWS 관련  강의를 유튜버인 Network Chuck 한테서 배웁니다.

l 다양한 자원 활용, 협력을 이끌어 내어 팀웍을 발휘하여 목표를 달성한 경험

저는 코딩부트캠프에서도 팀웍을 발휘하여 목표를 달성했지만, 그것보다 프리랜서로 일하면서 경험한 목표 달성 경험을 얘기해보고자 합니다. 프리랜서로 일을하면서 어느날 NFT에 관심이 높은 고객께서 저를 찾아오셔서 프로젝트를 시작했습니다.....(여기는 내가 알아서 수정할겡, 나머지만 봐줘잉~)

l 목표를 세우고 끈질기게 성취한 경험

이 주제에 관해서는 아무래도 초기에 HTML와 CSS 기본기를 배운후 클론코딩을 통해 저만의 작품을 만들었다는거에 매우 성취감을 느꼈습니다. 그 예시로는 카카오톡 UI/UX 클론코딩이 존재하며, 처음으로 느낀 성취감이었습니다. 또한 이 성취감에 취해 계속 다른 작품을 만들다 보니 React 를 만나게 되고 거기에 빠지게 되었습니다. 하지만 거기서 멈추기 싫었던 저는, 코딩부트캠프를 통해 미들웨어/백엔드 인 Java 를 만나게 되어 저의 코딩세계 시야를 더 넓혀 무긍무진한 가능성을 보게 되었습니다. 그렇기에 제 목표는 현재 제가 아는 언어와 새로운 언어, 새로운 알고리즘을 통해 더욱더 큰 목표를 세워, 그것을 성취해보고 싶습니다.
